Hello!

This is an automated program and will complete the process for you.


Instructions:

	1. Extract the zip file to your desktop 
	2. Add your images to the Image Files folder
	3. Wait for the process to complete


Requirements: ---------------------------------------------------------------------------------------------------------------------------------- 

1. Install Microsoft ICE
		64 bit - http://research.microsoft.com/en-us/downloads/69699e5a-5c91-4b01-898c-ef012cbb07f7/default.aspx
		32 bit - http://research.microsoft.com/en-us/downloads/730cd6bb-6450-4e66-8101-a94e71cb0779/default.aspx

2. Install c++/g++
		Windows - http://www1.cmc.edu/pages/faculty/alee/g++/g++.html
		Mac OS -  http://www1.cmc.edu/pages/faculty/alee/g++/g++_mac.html

3. Install OPENCV 
		Windows - http://sourceforge.net/projects/opencvlibrary/files/opencv-win/3.0.0/opencv-3.0.0.exe/download
		MAC OS -  https://github.com/Itseez/opencv/archive/3.0.0.zip

		Tutorial - https://www.youtube.com/watch?v=EcFtefHEEII
